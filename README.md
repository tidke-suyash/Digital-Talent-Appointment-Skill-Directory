# Faculty Appointment Console

**Made for MET (MET Institute of Management)**

## Overview
[cite_start]The Digital Talent Appointment System allows faculty to view student profiles and appoint individuals for academic tasks, projects, or responsibilities through a structured digital platform[cite: 2].

* [cite_start]Selection based on limited visibility[cite: 4].
* [cite_start]Manual communication[cite: 5].
* [cite_start]Lack of structured student skill records[cite: 6].
* [cite_start]Repeated dependency on same students[cite: 7].

## [cite_start]Proposed Solution & Features [cite: 8]
* [cite_start]Digital student profile cards[cite: 9].
* [cite_start]Display of skills and experience[cite: 10].
* [cite_start]Faculty selection through platform[cite: 11].
* [cite_start]Automatic code-generated message[cite: 12].
* [cite_start]Structured information capture through text fields[cite: 13].
* [cite_start]Organized tracking system[cite: 14].
* **System Capabilities:** Import/export `.xlsx` and `.csv` datasets, local browser storage, one-click academic year promotion, and a built-in dark/light mode toggle.

* [cite_start]Transparent selection process[cite: 17].
* [cite_start]Encourages skill development[cite: 18].
* [cite_start]Reduces manual coordination[cite: 19].
* [cite_start]Promotes merit-based appointments[cite: 20].

## Quick Setup
1. Clone this repository or download the HTML file.
2. Open the file directly in any modern web browser.
3. Import your student data spreadsheet to populate the directory.
4. Search, select, and assign tasks!
